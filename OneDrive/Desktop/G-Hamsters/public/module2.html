Модуль 2:
Крива Безьє

Мета модуля: Ознайомити слухачів з поняттям кривих Безьє, їх математичними основами, властивостями, способами побудови та практичним застосуванням у векторній графіці та комп'ютерному дизайні.

1. Вступ до Кривих у Комп'ютерній Графіці

У комп'ютерній графіці для зображення плавних ліній та складних форм використовуються математичні криві. Це дозволяє досягти:

  • Гнучкості та наглядності: Інтуїтивне керування формою.
  • Математичної простоти: Ефективні алгоритми для обчислень.
  • Універсальності: Широкий спектр застосувань, ефективне використання пам'яті.

Основні задачі при роботі з кривими:

  • Апроксимація: Заміна складної кривої на простішу, близьку за формою. Це спрощує обчислення, інтегрування, диференціювання.
  • Інтерполяція: Побудова кривої, що точно проходить через заданий набір точок. Можуть бути додаткові вимоги до гладкості.
  • Згладжування: Відновлення гладкої функції, що проходить поблизу заданих точок із допустимим відхиленням (часто для "зашумлених" даних).

Сплайни:
Один з поширених методів побудови кривих та поверхонь складної форми.

  • Це гладка крива, що будується з дуг і проходить через кілька контрольних точок, які керують її формою.
  • Математично, сплайн – це функція, область визначення якої розбита на сегменти, і на кожному сегменті функція є поліномом.

2. Криві Безьє: Історія та Застосування

Криві Безьє – це один з найважливіших типів сплайнів та фундаментальний інструмент векторної графіки.

Історична довідка:

  • 1959 р.: Поль де Кастельє (компанія "Сітроен") вперше використав схожі криві, але дослідження були комерційною таємницею.
  • 1960-ті рр.: П'єр Безьє (компанія "Рено") розробив та популяризував метод для комп'ютерного проектування кузовів автомобілів.
  • Згодом стали ключовим інструментом CAD-систем та програм комп'ютерної графіки.

Сфери застосування кривих Безьє:

  • Керування рухом об'єктів (анімація).
  • Створення анімаційних рисунків.
  • Опис художніх шрифтів (TrueType, PostScript).
  • Передача виділених об'єктів (контури, маски).
  • Інтерактивний дизайн форми різноманітних об'єктів.

3. Принцип Побудови та Термінологія Кривих Безьє

П'єр Безьє розробив метод, що створює природне сприйняття кривої для користувача.

  • Характеристична ламана (опорний багатокутник): Крива Безьє визначається набором контрольних точок, які утворюють відкритий багатокутник (ламану лінію). Ця ламана задає загальну форму кривої.
  • Контрольні точки:
    • Опорні точки (вузлові): Перша та остання точки характеристичної ламаної. Крива Безьє завжди проходить через ці точки.
    • Керуючі точки (напрямні): Проміжні точки характеристичної ламаної. Крива, як правило, не проходить через них, але вони визначають її вигин та напрямок дотичних у опорних точках.
  • Аналогія: Характеристичну ламану можна уявити як "металеву конструкцію", до якої крива Безьє "притягується як магнітна стрічка". Сила притягування до кожної точки визначає форму.

Змінюючи положення контрольних точок, користувач інтерактивно змінює форму кривої Безьє.

4. Способи Побудови Кривої Безьє

Існує три основні способи математичного опису та побудови кривих Безьє:

  1. Параметричний
  2. Рекурсивний (Алгоритм де Кастельє)
  3. Матричний
4.1. Параметричний вигляд (з використанням поліномів Бернштейна)

Крива Безьє степеня n визначається n+1 контрольною точкою P0, P1, ..., Pn.
Кожна точка на кривій B(t) для параметра t (де t змінюється від 0 до 1) обчислюється як:

B(t) = Σi=0..n [bi,n(t) * Pi]

де bi,n(t) – це поліноми Бернштейна:

bi,n(t) = C(n,i) * ti * (1-t)n-i

C(n,i) – біноміальний коефіцієнт (n! / (i! * (n-i)!)).

Приклади:

  • n=1 (Лінійна крива, 2 точки P0, P1): B(t) = (1-t)P0 + tP1 (відрізок прямої)
  • n=2 (Квадратична крива, 3 точки P0, P1, P2): B(t) = (1-t)2P0 + 2t(1-t)P1 + t2P2
  • n=3 (Кубічна крива, 4 точки P0, P1, P2, P3): Найчастіше використовується на практиці.
    B(t) = (1-t)3P0 + 3t(1-t)2P1 + 3t2(1-t)P2 + t3P3

Властивості поліномів Бернштейна:

  • Невід'ємні (bi,n(t) ≥ 0 для t в [0,1]).
  • Сума дорівнює одиниці (Σ bi,n(t) = 1).
  • Залежать лише від степеня n та індексу i, а не від координат точок Pi.
4.2. Рекурсивний вигляд (Алгоритм де Кастельє)

Геометричний метод побудови точки на кривій Безьє для заданого t:

  1. Кожна сторона початкової характеристичної ламаної ділиться у відношенні t : (1-t).
  2. Отримані точки поділу з'єднуються, утворюючи нову ламану з меншою на одну вершину кількістю сторін.
  3. Процес повторюється, доки не залишиться одна точка – це і є точка на кривій Безьє, що відповідає параметру t.
4.3. Матричний вигляд

Рівняння кривої Безьє можна записати у матричній формі: B(t) = T * M * G
Де:

  • T – вектор-рядок параметрів (наприклад, для кубічної: [t3 t2 t 1])
  • M – матриця базисних функцій (характеристична матриця, залежить від типу кривої)
  • G – вектор-стовпець контрольних точок (геометричний вектор)

Для кубічної кривої Безьє (n=3): 

B(t) = [t³ t² t 1] * | -1  3 -3  1 | * | P₀ |
                      |  3 -6  3  0 |   | P₁ |
                      | -3  3  0  0 |   | P₂ |
                      |  1  0  0  0 |   | P₃ |

(Порядок рядків/стовпців у матрицях може варіюватися в літературі, але результат той самий).

5. Властивості Кривих Безьє

  1. Форма: Крива повторює загальний хід характеристичної ламаної. Порядок контрольних точок суттєво впливає на форму.
  2. Кінцеві точки: Крива завжди починається в точці P0 (B(0) = P0) і закінчується в точці Pn (B(1) = Pn).
  3. Дотичні в кінцевих точках:
    • Дотична до кривої в початковій точці P0 збігається з напрямком відрізка P0P1.
    • Дотична до кривої в кінцевій точці Pn збігається з напрямком відрізка Pn-1Pn.
  4. Інваріантність щодо афінних перетворень: Афінне перетворення кривої (поворот, масштабування, перенесення) еквівалентне виконанню цього ж перетворення над її контрольними точками з подальшою побудовою кривої. Це спрощує маніпуляції.
  5. Властивість опуклої оболонки: Крива Безьє завжди повністю лежить всередині опуклої оболонки, утвореної її контрольними точками. Це корисно для тестів на перетин та відсікання.
  6. Відсутність локального контролю (для однієї кривої): Зміна положення будь-якої однієї контрольної точки впливає на форму всієї кривої.
  7. Залежність степеня від кількості точок: Степінь полінома, що описує криву, на одиницю менший за кількість контрольних точок. Додавання точки підвищує степінь і потребує перерахунку.

6. Складені Криві Безьє

Для створення складних форм та забезпечення локального контролю окремі сегменти кривих Безьє (зазвичай кубічних) з'єднують між собою.

  • Побудова: Загальний набір точок розбивається на підмножини (наприклад, по 4 точки для кубічних сегментів), де остання точка попереднього сегмента є першою точкою наступного.
  • Неперервність (C0): Забезпечується автоматично, якщо кінцева точка одного сегмента збігається з початковою точкою наступного.
  • Гладкість (C1): Для забезпечення гладкості в точці з'єднання двох кубічних сегментів (неперервності першої похідної), три точки – остання керуюча точка першого сегмента, спільна опорна точка, та перша керуюча точка другого сегмента – повинні лежати на одній прямій.

Використання складених кривих дозволяє змінювати форму одного сегмента, не впливаючи на інші, тобто забезпечує локальний контроль.

7. Переваги та Недоліки Кривих Безьє

Переваги:
  • Інтуїтивність: Легкість керування формою через контрольні точки.
  • Математична простота: Відносно прості формули для обчислень.
  • Обчислювальна стійкість: Алгоритми побудови є стійкими.
  • Гарантована гладкість (для одного сегмента).
  • Властивість опуклої оболонки спрощує багато алгоритмів (наприклад, відсікання, перетин).
  • Інваріантність до афінних перетворень.
Недоліки (для окремої, не складеної кривої):
  • Глобальний контроль: Зміна однієї контрольної точки впливає на всю криву.
  • Степінь залежить від кількості точок: Для великої кількості точок степінь кривої стає високим, що ускладнює обчислення та може призводити до небажаних коливань.
  • Додавання/видалення точок змінює степінь і вимагає повного перерахунку кривої.

Ці недоліки значною мірою нівелюються використанням складених кривих Безьє.
